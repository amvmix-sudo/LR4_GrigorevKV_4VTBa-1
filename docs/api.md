# Описание API

В рамках данной лабораторной работы приведено описание примеров API.
Описание носит демонстрационный характер и предназначено для учебных целей.
Реализация API в рамках работы не требуется.

API предназначено для взаимодействия с системой извне и позволяет получать
информацию о пользователях, а также добавлять новых пользователей.

---

## Получение списка пользователей

Данный метод предназначен для получения списка всех пользователей системы.

**Метод:** GET  
**URL:** `/api/users`

### Описание ответа

В ответе возвращается список пользователей в формате JSON.

### Пример ответа

```json
[
  {
    "id": 1,
    "username": "user1",
    "email": "user1@example.com"
  },
  {
    "id": 2,
    "username": "user2",
    "email": "user2@example.com"
  }
]
